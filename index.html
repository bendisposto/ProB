Installing the ProB plug-ins

Please use the Rodin update manager to install the plug-ins.

  1. Select from main menu: Help | Install New Software...
  2. Under Work with, select the ProB repository. If it isn't listed, add a new repository using the URL
    https://stups.hhu-hosting.de/rodin/prob1/release/
  3. Select the plug-ins you want to install and follow the dialog.
